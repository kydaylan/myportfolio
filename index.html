How I Work with AI

Three of the six case studies above involve AI-assisted development. Here is how that works in practice:

I design the solution. AI helps build it.

I identify the problem, design the architecture, and write detailed technical specifications. I then use tools like Claude Code to develop to that spec, and Google AI Studio for image generation. The strategic thinking, the specification writing, and the quality control are mine. The code generation and image rendering are the execution method. Every project is tested, iterated, and deployed by me across live production environments.
